SimpleStorage - this demo shows how easy you can create/save/load tree storage.
D2009+

Everything what is important for this demo you can find in uTypes.pas
Code in implementation is generally not need, so only what you need
it to define pure record/dynamic array structures and ... that's all.
Save and load of any defined structure is performed only by single line of code,
see TSimpleTreeStorage.LoadFrom/SaveTo
